# Turtle Graphics - Ejercicios de Gráficos en Python

Colección de ejercicios educativos utilizando la librería Turtle de Python para crear gráficos y figuras geométricas.

## Descripción

Turtle Graphics es una herramienta educativa que permite crear gráficos y figuras geométricas mediante comandos simples. Imagina una "tortuga" virtual que puedes programar para dibujar en la pantalla.

## Contenido

### 📐 Figuras Geométricas
- **cuadrados.py** - Programa para dibujar uno o múltiples cuadrados
- **regpoly.py** - Polígonos regulares genéricos
- **regpoly-4.py** - Polígonos de 4 lados (cuadrados)
- **regpolydef-4.py** - Polígonos de 4 lados con parámetros definidos

### 🌀 Patrones y Espirales
- **espirales.py** - Múltiples espirales concéntricas
- **espiral-creciente.py** - Espiral de tamaño creciente
- **petalos.py** - Patrón de pétalos y flores

## Requisitos

- Python 3.x
- Librería Turtle (incluida en Python estándar)

## Uso

1. Asegúrate de tener Python instalado en tu sistema
2. Abre una terminal o línea de comandos
3. Navega al directorio con los archivos
4. Ejecuta un script con: `python nombrearchivo.py`
5. Se abrirá una ventana gráfica mostrando el dibujo

### Ejemplo:
```bash
python cuadrados.py
```

## Características Educativas

- **Programación de gráficos**: Aprende cómo controlar gráficos mediante código
- **Geometría**: Practica conceptos de geometría mediante visualización
- **Bucles y condicionales**: Utiliza bucles para generar patrones
- **Funciones**: Organiza el código en funciones reutilizables
- **Ángulos y trigonometría**: Comprende rotaciones y posicionamiento

## Conceptos Clave

- **turtle.forward(distance)** - Mueve la tortuga hacia adelante
- **turtle.right/left(angle)** - Gira la tortuga a derecha o izquierda
- **turtle.penup/pendown()** - Levanta o baja el lápiz
- **Bucles for** - Para crear patrones repetitivos
