# Programación Python - Proyectos Educativos

Colección de proyectos educativos de programación en Python, incluyendo robótica con GiggleBot, programación de Microbit y gráficos con Turtle.

## Contenido del Repositorio

Este repositorio contiene tres carpetas principales con diferentes tipos de ejercicios y proyectos:

### 📁 [GIGGLEBOT](./GIGGLEBOT/)
Proyecto de robótica educativa con el robot GiggleBot. Incluye tres fases progresivas:
- **Fase 1**: Ejercicios fundamentales de movimiento y control del GiggleBot
- **Fase 2**: Búsqueda y localización de balizas, recogida de secretos
- **Fase 3**: Trabajo en equipo con arquitectura Jefe-Esclavo

### 📁 [MICROBIT](./MICROBIT/)
Ejercicios de programación para la plataforma BBC Micro:bit, incluyendo:
- Manejo de botones y sensores
- Visualización en pantalla LED
- Brújula digital
- Juegos interactivos
- Control de morse

### 📁 [Turtle](./Turtle/)
Ejercicios de gráficos con la librería Turtle de Python:
- Figuras geométricas
- Spirales y patrones
- Figuras regulares

## Requisitos

- Python 3.x
- Para MICROBIT: Mu Editor o similar
- Para GIGGLEBOT: Hardware GiggleBot compatible
- Para Turtle: Librería turtle (incluida en Python estándar)

## Uso

Navega a cada carpeta para más información específica sobre los proyectos incluidos en cada una.
