# Fase 1 - Ejercicios Fundamentales

## Descripción

Fase 1 del proyecto GiggleBot con siete ejercicios progresivos que cubren los fundamentos de la programación de robótica con el GiggleBot.

## Ejercicios Incluidos

1. **gigg_EJ1.py** - Movimiento básico
2. **gigg_EJ2.py** - Control de motores
3. **gigg_EJ3.1.py** - Sensores (Primera parte)
4. **gigg_EJ3.2.py** - Sensores (Segunda parte)
5. **gigg_EJ4.py** - Interacción con el entorno
6. **gigg_EJ5.py** - Programación modulada
7. **gigg_EJ6.py** - Proyecto integrador

## Objetivos

- Comprender el funcionamiento básico del GiggleBot
- Controlar motores y movimiento
- Leer sensores y responder a estímulos
- Estructurar código en funciones reutilizables
- Integrar movimiento y sensores en un proyecto

## Requisitos

- 1 GiggleBot con Microbit
- Mu Editor
- Pilas en buen estado

## Instrucciones

1. Carga cada programa en el Microbit usando Mu Editor
2. Instala el Microbit en el GiggleBot
3. Ejecuta y observa el comportamiento
4. Modifica y experimenta según tus necesidades

## Notas

- Los ejercicios están diseñados para ejecutarse en orden progresivo
- Cada ejercicio incrementa la complejidad de forma gradual
- Los ejercicios pueden servir como base para proyectos personalizados
